# Transplant Workflow Engine

This skill covers the 13-stage workflow, decision types, routing logic, and auto-advancement rules.

## Read First

- `docs/HANDOFF.md`
- `references/stages.md`
- `references/hard-gates.md`

## Rules

- Preserve hard gates
- Preserve explicit routing and ownership rules
- Avoid hidden workflow transitions
- Keep auto-generated decisions auditable
- Prefer deterministic logic for stage advancement

## Decision Types

| Decision Type | Made By | Possible Outcomes | Triggers |
|--------------|---------|-------------------|----------|
| ie-review | front-desk | proceed | Confirms I/E answers are coherent |
| initial-screening | front-desk | eligible, flagged-for-senior | Deterministic + edge cases |
| senior-disposition | senior-coord | proceed, clarify, not-appropriate | After flagged |
| financial | financial-coord | cleared, pending, issue | Insurance verification |
| document-validation | various | valid, invalid, expired | Each uploaded doc |
| specialist-review | specialist | cleared, clarification, escalate | Dietitian / SW / Nephrology |
| final-decision | senior-coord | approved, more-info, not-approved | After all specialists |
| no-response-3x | system | needs-senior-review | Auto-created at 3 attempts |
| end-referral | senior-coord | ended | Any time |

## Contact Attempt Escalation

```typescript
function syncContactAttemptEscalation(caseId: string, attemptCount: number) {
  if (attemptCount >= 3) {
    createDecision({
      caseId,
      decisionType: 'no-response-3x',
      outcome: 'needs-senior-review',
      autoGenerated: true
    });
  }
}

## Auto-Advancement Pattern

```typescript
function maybeAdvanceCaseStage(caseId: string, currentStage: Stage) {
  const advancementRules = {
    'patient-onboarding': () => allRoisSigned(caseId),
    'initial-todos': () =>
      allTodosComplete(caseId) && ieConfirmReviewComplete(caseId),
    'records-collection': () =>
      allRequiredDocsValidated(caseId) && has2728Validated(caseId),
  };

  if (advancementRules[currentStage]?.()) {
    transitionTo(nextStage(currentStage));
  }
}
