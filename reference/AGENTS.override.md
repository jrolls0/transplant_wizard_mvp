- Everything under `reference/` is read-only.
- Never modify files in this subtree.
- Use these files for patterns, architecture understanding, and UI inspiration only.
- Copy ideas into real implementation files under `src/` or `supabase/`; do not edit reference files directly.

- The patient mobile reference is for mobile UI design direction only.
